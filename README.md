# Portfólio Fabiana De Brito

Portfólio profissional de Fabiana De Brito, criado com HTML, CSS e JavaScript puro.

## Sobre

O projeto apresenta uma página única, direta e responsiva, com:

- Apresentação profissional.
- Resumo sobre a transição para Salesforce.
- Tecnologias com as quais tive contato.
- Soft skills.
- Espaço para projetos em construção.
- Download do currículo em PDF.
- Links para LinkedIn, GitHub, Trailblazer e e-mail.

## Tecnologias

- HTML
- CSS
- JavaScript

## Como Visualizar

Abra o arquivo `index.html` no navegador.
